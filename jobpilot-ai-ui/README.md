# JobPilot AI UI

A new Next.js + Tailwind UI for JobPilot AI.

## Setup

1. `cd jobpilot-ai-ui`
2. `npm install`
3. `npm run dev`

## Features

- Landing page with customer-focused sections
- Auth pages for sign in / sign up / password reset
- Dashboard shell with agent, resume builder, ATS, and reporting modules
- Modern components using Tailwind, Framer Motion, and Lucide icons
